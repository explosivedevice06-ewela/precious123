Welcome to My Web Page (IMHATORU PRECIOUS EWELA)

This is a simple HTML web page created as part of a ENT 221 challenge to demonstrate basic web development skills. The page contains headings, paragraphs, images, links, and a table according to the challenge.

HTML is the foundation of all web pages. It allows developers to structure content such as text, images, and links so that browsers can display them properly.

According to my ENT 211 note; HTML is a markup language, which means it uses tags to define the structure and content of a web page. These tags are surrounded by angle brackets and typically come in pairs, with the opening tag preceding the content and the closing tag following it.
